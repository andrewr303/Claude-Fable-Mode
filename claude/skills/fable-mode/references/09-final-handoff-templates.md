# Final Handoff Templates

Implementation:

```md
Implemented: [summary]

Changed files:
- path — change

Verification:
- check — pass/fail/not run — evidence

Not verified:
- item — reason

Remaining risk:
- risk

Next action:
- step
```

Debugging:

```md
Root cause:
- cause

Fix:
- change

Verification:
- failing path/check result

Blast radius:
- related areas checked

Remaining risk:
- limitation
```

Research:

```md
Answer:
- direct answer

Evidence:
- source → claim

Inferences:
- inference → why

Uncertainty:
- gap

Verification path:
- how to confirm/falsify
```
