# Security Policy

Do not submit secrets, tokens, credentials, private keys, proprietary hidden prompts, or private user data in issues, discussions, pull requests, or examples.

For security-sensitive reports, open a private advisory if available in the hosting repository, or contact the repository maintainer directly.

This project is an instruction and template package. It should not be used to bypass platform policy, hidden prompts, model safeguards, production approval gates, or human review for high-impact actions.
