# Source Inventory

This public package was synthesized from several categories of material:

- Fable Mode / Fable Method artifacts supplied by the project owner.
- Public-style agentic engineering principles: staged planning, tool contracts, approval gates, observability, run records, evals, and prompt-injection hygiene.
- Software engineering fundamentals: codebase archaeology, small reversible changes, testing discipline, production readiness, and readable handoffs.
- TypeScript engineering practices: avoiding unsafe `any`, using `unknown` for untrusted input, modeling valid states, exhaustiveness checks, and public API type hygiene.
- Multi-agent system practices: planner/executor/verifier separation, strict handoff schemas, scatter-gather research, and human gates for irreversible actions.

Project-specific product content from earlier internal drafts was intentionally removed. The published files are project-neutral and include customization slots for users to add their own facts.
