# Fable Mode

Fable Mode is a project-neutral operating standard for agentic coding tools. It includes drop-in `AGENTS.md` and `CLAUDE.md` files, Claude Code agents and skills, and a Codex plugin.

Use it when a task is too risky or complex for one-shot code generation: multi-file changes, debugging, AI/RAG/tooling work, migrations, production operations, generated artifacts, or anything that needs proof instead of confidence.

Start with the repository README, then install the assets that match your tool.
